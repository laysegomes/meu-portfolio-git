 Registro de Resolução de Conflito

O que causou o conflito:
O conflito ocorreu no arquivo `REGISTRO.md` porque eu tentei juntar a `branch-b` na `main` após a `branch-a` já ter feito uma alteração na mesma linha. Ambas as branches tentaram definir uma "versão" diferente para o mesmo trecho de texto.

Como decidi qual versão manter:
Eu escolhi manter ambas as versoes, pois era a alteração mais recente e completava o histórico do exercício.

Comando usado:
Para resolver, usei o comando `git merge` no terminal, abri o VS Code para escolher a versão final, salvei e finalizei com `git add` e `git commit`.
